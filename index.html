My Personal Projects

Timer for Rubik's Cube

View project on GitHub
View project on the App Store

Timer for Rubik's Cube is my first iOS app on the App Store. I developed the app to log and track 3x3 Rubik's Cube solves. A random scramble is generated every solve.

The app has support for inspection time that can be adjusted from 0 seconds to a minute.

It stores your times so that you can view them at a later date, as well as giving you the most important statistics about your solves in general. You can email your times to yourself so you have a list of times available on your computer.

Workathon

View project on GitHub

Workathon is a Google Chrome extension that blocks distracting websites in the hope that you will be more productive. You can add websites to a list of sites that will be blocked.

You can then select the times of the day that you want the sites to be blocked.

The extension was written in JavaScript, with the interface done in HTML and CSS. This was also the first hackathon project I worked on, at UofTHacks.

Recreation of Terraria

View project on GitHub

This project was made in Grade 11 by Calvin Nhieu and I. We wanted to get better at developing games, and decided that the best way to do that would be to recreate a game called Terraria. Terraria is a 2D side-scroller game similar to Minecraft. We stopped working on this project after most of the early features were developed.

The game itself was developed in Java using Eclipse (I have since moved onto IntelliJ for Java development :P). The game engine was written from scratch.

Some aspects we particularly enjoyed working on were the terrain generation and the enemy artificial intelligence.
